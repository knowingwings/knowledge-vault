# Pre-Reading Scaffolding Prompt

```
I'm about to read [paper title] on [topic]. Give me:
1. The 5 key concepts I'll encounter and one-sentence definitions
2. The relationships between these concepts
3. What prerequisite knowledge I need
4. What questions this material is trying to answer

Be concise—this is orientation, not a substitute for reading.
```

## Usage
Use before reading a complex paper or chapter to build mental scaffolding.
