# Anki Card Generation Prompt

```
Read [[Note title]] and generate 5 Anki cards testing the key concepts. Format as markdown table with Front/Back.
```

## Usage
Use to create spaced repetition cards from permanent notes.

## Good Card Principles
- One atomic fact per card
- Include context (when/why would I use this?)
- Add visual diagrams for spatial concepts
- Include "boundary" cards: "When does this NOT apply?"
