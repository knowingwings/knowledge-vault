## Session Templates

### Calisthenics Session (~30-45 min)

**Warm-up (5 min)**
- Weighted jump rope: 2-3 min
- Dynamic stretches

**Main Circuit (20-30 min)** - 3 rounds
- Push ups (push up bars): 10-15 reps
- Ab roller: 8-12 reps
- Resistance band rows: 12-15 reps
- Resistance band squats: 15 reps
- Plank hold: 30-60 sec
- Rest: 60 sec between rounds

**Finisher (5-10 min)**
- Weighted jump rope: 3-5 min intervals
- Core work: leg raises, bicycle crunches

**Cool-down (5 min)**
- Static stretches on yoga mat

### Plyometrics Session - Tuesday (~30-40 min)

**Warm-up (5 min)**
- Light jump rope
- Dynamic stretches
- Leg swings

**Main Work (20-25 min)**
- Box jumps (or tuck jumps): 3x8
- Burpees: 3x10
- Jump squats: 3x12
- Lateral bounds: 3x10 each side
- Jump lunges: 3x8 each leg
- Rest: 90 sec between sets

**Cool-down (5-10 min)**
- Light stretching
- Foam rolling if available
