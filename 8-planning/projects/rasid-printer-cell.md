# RASID: Autonomous 3D-Printer Unloader Cell

Hub for the MSc RASID project: an autonomous robotic cell (SCARA plus a custom vertical rail) that detaches a finished print bed from a printer and racks it.

**Headline.** 90.3 s hardware-in-the-loop cycle vs a 120 s target. Custom C++ ros2_control hardware interface; synthetic-data YOLOv8n-pose vision; the physical end-effector was the only hardware element.

**Graded.** Module 75.9% (report 80%).

**Reuse.** Flagship project in the job applications and portfolio. See [[2026-04-22-isam-job-targets]].
